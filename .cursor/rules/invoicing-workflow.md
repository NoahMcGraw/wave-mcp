# Invoicing Workflow Config

Before processing payments, sending receipts, or performing any repetitive invoicing task for Hypernova USA, read `invoicing-config.json` in the project root. It contains all saved defaults (IDs, accounts, email templates, etc.) and a step-by-step workflow so you don't need to look anything up.
